Spiel starten

Der PaziFist ist freie Software und steht unter der GPL. Der Quellcode kann über ein Github Repository eingesehen werden.

Das Spiel ist in Java geschrieben. Um es ausführen zu können benötigen Sie also eine geeignete Version der Java Runtime Environment.

Sollten Sie Probleme haben, das Spiel über den obigen Link zu starten, steht auch ein ausführbares Jar-Archiv zum direkten Download zur Verfügung.

Unter bestimmten Versionen von Mac OS X, auf denen Java 7 installiert ist, kann es zu einem Absturz der Java Virtual Machine kommen. Die Ursache hierfür ist ein Problem von LWJGL, Der PaziFist ist also für diesen Fehler nicht selbst verantwortlich. Bis dieser Fehler behoben ist, steht eine angepasste Version des PaziFisten zur Verfügung, die auf die Verwendung von LWJGL verzichtet. Leider ist in dieser Version allerdings die Darstellung deutlich langsamer und es kann zu Problemen bei der Tastatureingabe kommen. Um das ausführbare Jar-Archiv auf einem solchen System benutzen zu können, starten Sie dem Programm das Argument -noglview.
